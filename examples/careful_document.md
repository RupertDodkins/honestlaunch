# Careful Scientific Claim Demo

## Abstract

We evaluate CappinCheck on a curated benchmark of technical-document claims. CappinCheck improved from 84.1% to 87.3% on Benchmark X, a 3.2 percentage-point gain under the benchmark conditions.

## Results

| System | Benchmark X accuracy |
| --- | ---: |
| Baseline | 84.1% |
| CappinCheck | 87.3% |

## Limitations

The result is limited to the curated Benchmark X dataset. The evaluation does not establish real-world deployment performance, legal-document performance, clinical-document performance, or user preference.
